## 1. note への貼り付け手順

### 1.1 記事タイトル (note 上の表示タイトル)

note のタイトル欄に以下を貼る:

```
顔出しなしで30秒、シネマ質感の縦型動画。¥15,000 / 本。
```

- 全角で 34 文字。note の検索結果・SNS OGP で切れない長さに収めた
- **サブタイトル機能は使わない** (note の仕様上、本文 1 行目が概要に入るため)

### 1.2 本文

`ArcoCapital/SNS動画運用事業部/営業/LP.md` の **2 行目以降** (`# 顔出しなしで...` の見出し以下、ファイル全体) をコピーし、note の本文に貼り付ける。

**注意**:
- note は Markdown の表 (`|---|`) を部分的にしか描画しません。3つの商品セクションの「価格・納期・用途 一覧」と「納品フロー」の表は、貼り付け後に **note エディタ上で「表」ブロックとして再作成** するか、手動で整形する
- 免責段落が H2 にならないよう、note エディタ上でも見出しスタイルを外す (preference_no_fake_branding.md 準拠)
- 本文内のバッククォート `` ` `` は note では標準の等幅フォント扱いになるため原則そのままで OK

### 1.3 タグ (最大 5 個)

note タグ欄に以下を **順番どおり** 貼る:

1. `縦型動画制作`
2. `Reels`
3. `小規模事業者`
4. `AI動画`
5. `制作代行`

(サブ: `シネマティック` `顔出しなし` は余白があれば追加可)

- サイズ: 1280×670 (note 推奨)

※ サムネが用意できない場合は、本文冒頭の絵コンテプレビューだけで公開可。

- **有料/無料**: 無料 (全文公開)
- **マガジン**: 設定しない (1 本完結の LP)

## 4. 失敗時の撤回手順

公開後に致命的誤記が見つかった場合:

1. **即時**: note 編集画面で該当箇所を修正 (公開 URL は保持される)
2. **軽微**: 日次更新として `LP 更新履歴` セクションを LP.md 末尾に追記する (v3 として変更点記録)
3. **重篤 (炎上誘発)**: note 記事を非公開に戻し、`operations/incident_log.jsonl` に記録、COO へ報告

**撤回基準**:
- 差別的・誤解を招く表現
- 対象外ペルソナへの明示的批判
- 法令違反を疑わせる表現 (景表法・特商法など)
