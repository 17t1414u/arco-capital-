## 3. ガードレール連携

各 DivisionCrew の `__init__` は起動時に `operations/guardrails.yaml` を読み、
現在モード (A/B/C) を ModeManager 経由で判定する:

- **Mode A**: 外部送信・有料クレジット無効の旨を標準出力に警告
- **Mode B**: オーナー承認ゲートが必要である旨を標準出力に通知
- **Mode C**: 自動実行モードを明示

Task の description 内には予算ハードキャップ (¥10k/週 Renoise / ¥1.5k/日 API) を明記しており、
超過時は `operations.budget_tracker.BudgetBreach` 例外 → `kill_switch.py` 発火という静かな失敗禁止フロー。
