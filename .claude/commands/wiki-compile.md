# /wiki-compile — Wiki全体の再構築・整理

`wiki/` 全体を見渡してページの整合性を確認し、インデックスと概念間リンクを再構築します。

## 手順

1. `C:/Users/17t14/Documents/Obsidian Vault/CLAUDE.md` を読んでスキーマを確認
2. `wiki/sources/` の全ソース要約を読み込み
3. 概念の出現頻度を集計
   - 2ソース以上 → `wiki/concepts/` にフルページが存在するか確認。なければ作成
   - 1ソースのみ → スタブとして記録
4. エンティティページの網羅性を確認
5. `wiki/syntheses/` に必要な統合分析を作成・更新
6. `wiki/index.md` を全ページリストから再生成
7. `wiki/log.md` に操作記録を追記

## 用途

- 大量のIngest後に一度Wikiを整理したいとき
- 構造が散らかってきたと感じたとき
- 定期メンテナンス（月1回程度推奨）
